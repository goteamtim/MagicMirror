Magic Mirror

Express app that powers magic mirror software

Welcome to the home of the Magic Mirror!

This project was designed with the idea that someone without a lot of technical experience could download this code, drop it on a raspberry pi, hook up a monitor, and have a magic mirror. A tutorial on how to set this up is coming soon. Changes to make the mirror custom are handled by a setup page so you don't have to dive into too much code.

Customize

The idea is to be able to add elements and remove them as you see fit. If you have a suggestion please feel free to fork but let me know as well! I would love to see what you came up with.

Authors and Contributors

Right now I (@goteamtim) am the only contributor but would love some help. Feel free to reach out if you want to help out. I happily welcome someone else to learn with me.
